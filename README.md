# 🍳 Recipe Page

A simple and responsive recipe page built using HTML and CSS.

## 📌 Overview

This project was created as part of a Frontend Mentor challenge. The goal was to recreate a recipe page design as closely as possible using clean code and responsive layout techniques.

## 🚀 Features

* Responsive design (works on mobile and desktop)
* Clean and structured HTML
* Styled with CSS
* Organized layout with sections (ingredients, instructions, etc.)

## 🛠️ Built With

* HTML5
* CSS3
* Flexbox / basic layout techniques
